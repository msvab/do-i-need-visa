# Do I need visa?

Sample Rails app used to learn Ruby and Rails

Demo running on https://do-i-need-visa.herokuapp.com/ or admin site on https://do-i-need-visa.herokuapp.com/admin/visa_sources.

Technologies used:

* Ruby 2.2.0
* Rails 4.2
* Postgres
* Clearance
* SASS
* Bootstrap

* RSpec
* Capybara
* WebMock
